XML2RFC Creation Wizard



This XML2RFC Creation Wizard is designed to help you create a new internet draft using the XML2RFC XML directives. The XML is constantly displayed at the bottom, and all changes are immediately reflected there.

In the following pages, you will be asked a series of questions about your draft.
